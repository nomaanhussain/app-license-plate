## Setup

## Installation

Clone repository

Install the dependencies and devDependencies and start the server.

```sh
cd app-license-plate
pip install -r requirements.txt
```

For setting db...

```sh
set FLASK_APP=api
flask db init
flask db migrate
flask db upgrade
```

## Docker

 App License Plate is very easy to install and deploy in a Docker container.


```sh
cd app-license-plate
docker-compose build
```

This will create the image and pull in the necessary dependencies.

Once done, run the Docker Compose image

```sh
docker-compose up -d
```
Verify the deployment by navigating to your server address in
your preferred browser.

```sh
http://127.0.0.1:5000/
```
